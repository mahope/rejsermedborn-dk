## Strandrejse med børn kræver ekstra pakning

En strandrejse er fantastisk for familier – men det kræver lidt ekstra planlægning at pakke rigtigt. Denne pakkeliste sikrer, at I har alt med til sol, sand og vandleg.

## Sol- og strandbeskyttelse

### Solcreme og UV-beskyttelse

- [ ] Solcreme faktor 50+ (børn)
- [ ] Solcreme faktor 30-50 (voksne)
- [ ] Vandfast solcreme til badning
- [ ] Læbepomade med SPF
- [ ] After sun lotion
- [ ] UV-dragt til børn (50+ beskyttelse)
- [ ] UV-badebleer (hvis baby)

**Pro tip:** Påfør solcreme hjemmefra om morgenen – det giver bedre beskyttelse end at smøre på stranden.

### Hovedbeklædning

- [ ] Solhat med nakkebeskyttelse (børn)
- [ ] Kasket eller hat (voksne)
- [ ] Buff eller bandana (ekstra beskyttelse)

### Solbriller

- [ ] Solbriller til børn (med snor så de ikke mistes)
- [ ] Solbriller til voksne
- [ ] Polariserede glas anbefales

## Strandudstyr

### Strand-essentials

- [ ] Strandtæppe eller strandmåtte
- [ ] Parasol eller strandtelt (UV-beskyttelse)
- [ ] Køletaske til mad og drikke
- [ ] Vandflasker (genopfyldelige)
- [ ] Strandstole (hvis plads i bagagen)

### Sandlegetøj

- [ ] Spade og skovl
- [ ] Sandformer og spande
- [ ] Sandsi (til skattejagt)
- [ ] Legetøjsbiler til sandet
- [ ] Vandpistol eller vandballon-sprøjte

### Vandlegetøj

- [ ] Svømmevinger (godkendte!)
- [ ] Badering eller poolnudler
- [ ] Dykkerbriller og snorkel
- [ ] Vandsko (beskytter mod varmt sand og sten)
- [ ] Strandboldt
- [ ] Oppustelig bademadras

## Badetøj

### Til børn

- [ ] Badetøj x 3-4 (tørrer mellem brug)
- [ ] UV-dragt/rashguard
- [ ] Våddragtsshorts (hvis koldt vand)
- [ ] Badeble (baby/småbørn)

### Til voksne

- [ ] Badetøj x 2-3
- [ ] Sarong eller strandkjole
- [ ] Sandaler til strand
- [ ] Hurtigt-tørrende shorts

## Praktisk strandudstyr

### Opbevaring og organisering

- [ ] Vandtæt taske til mobil og værdigenstande
- [ ] Netpose til sandlegetøj (sandet falder ud)
- [ ] Plastikpose til vådt tøj
- [ ] Lille rygsæk til strand-essentials

### Hygiejne

- [ ] Vådservietter (sand på fingre!)
- [ ] Håndklæder (microfiber tørrer hurtigt)
- [ ] Talcumpudder (fjerner sand fra huden)
- [ ] Lille førstehjælpssæt

### Mad og drikke

- [ ] Snacks (tåler varmen: kiks, frugt, nødder)
- [ ] Frugt i skiver
- [ ] Juice eller vand (masser af væske!)
- [ ] Kølelementer til køletasken
- [ ] Plastikbestik og -tallerkener

## Tøj til strandrejse

### Daglig påklædning (pr. barn)

- [ ] T-shirts x 7+ (sved og solcreme kræver hyppige skift)
- [ ] Shorts x 5-6
- [ ] Lette kjoler (piger) x 3-4
- [ ] Underbukser x 8+ (ekstra!)
- [ ] Lettest mulige sko/sandaler
- [ ] 1 par lukkede sko (til udflugter)
- [ ] Let cardigan til aftener
- [ ] Nat-tøj

### Voksne

- [ ] Lette toppe/skjorter
- [ ] Shorts og nederdele
- [ ] Strandkjoler
- [ ] Aften-outfit (1-2)
- [ ] Let jakke/cardigan

## Specielt for babyer

### Strand med baby

- [ ] Pop-up strandtelt med UV-beskyttelse
- [ ] Oppusteligt babybassin
- [ ] Blødt strandunderlag
- [ ] Ekstra bleer
- [ ] Skifteunderlag
- [ ] Paraply til skygge over barnevogn
- [ ] Vand til at skylle sand af

### Mad til baby

- [ ] Modermælkserstatning (hvis relevant)
- [ ] Snacks tilpasset alder
- [ ] Skefuld/kop
- [ ] Kølebeholder til mad

## Underholdning

- [ ] Kortspil (vandtætte)
- [ ] Rejsespil

### Til eventuelle regnvejrsdage

- [ ] Puslespil
- [ ] Brætspil i rejseudgave
- [ ] Bøger
- [ ] Download film/serier til offline

## Tips til smart strandpakning

### Rullemetoden

Rul tøjet i stedet for at folde. Det sparer plads og minimerer folder.

### Pakkeposer

Brug pakkeposer eller compression bags:
- En pose til hvert familiemedlem
- En pose til badetøj
- En pose til beskidt tøj

### Dobbelt op på solcreme

Solcreme er dyrt på destinationen. Pak rigeligt!

### Tjek bagageregler

Flydende produkter (solcreme, after sun) skal i indtjekket bagage hvis over 100ml.

### Vask på ferien

Book overnatning med vaskemaskine – så kan I pakke halvt så meget tøj.
