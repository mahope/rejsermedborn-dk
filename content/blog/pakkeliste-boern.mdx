## Grundlæggende pakkeliste til alle aldre

Uanset om I rejser med baby eller teenager, er der nogle ting der altid skal med.

### Dokumenter og økonomi

- [ ] Pas (tjek udløbsdato!)
- [ ] Sygesikringskort / EU-sygesikringsbevis
- [ ] Rejseforsikringsdokumenter
- [ ] Kopi af pas (digital + fysisk)
- [ ] Betalingskort og lidt kontanter
- [ ] Bookingbekræftelser (hotel, fly, bil)

### Medicin og førstehjælp

- [ ] Receptpligtig medicin
- [ ] Panodil/Ipren (børnedosering)
- [ ] Solcreme (høj faktor)
- [ ] Myggespray (børnevenlig)
- [ ] Plaster og sårdesinficering
- [ ] Termometer
- [ ] Eventuel reisesygemedicin

## Baby og småbørn (0-3 år)

### Mad og amning

- [ ] Modermælkserstatning (hvis relevant)
- [ ] Sutteflasker
- [ ] Hagesmække
- [ ] Snacks (riskager, frugtmos)
- [ ] Rejse-højstol eller stol-sele

### Søvn

- [ ] Rejseseng eller foldeseng
- [ ] Yndlingsbamse eller sutteklud
- [ ] White noise-maskine eller app
- [ ] Mørklægningsgardiner (portable)
- [ ] Vant sengetøj eller lagen

### Bleskift og hygiejne

- [ ] Bleer (nok til rejsedagen + første dage)
- [ ] Vådservietter
- [ ] Skiftunderlag
- [ ] Blenumscreme
- [ ] Baby-shampoo og sæbe

### Transport

- [ ] Autostol (tjek reglerne i destinationslandet)
- [ ] Klapvogn eller barnevogn
- [ ] Bæresele
- [ ] Solskærm til klapvogn

## Børn 4-10 år

### Underholdning

- [ ] Tablet med downloadede film/spil
- [ ] Hovedtelefoner (børnevenlige med volumengrænse)
- [ ] Tegnesager og farver
- [ ] Små legetøj og biler
- [ ] Bøger eller lydbøger
- [ ] Rejsespil

### Tøj (beregn 1 sæt pr. dag + ekstra)

- [ ] T-shirts
- [ ] Shorts/bukser
- [ ] Badetøj (2 sæt)
- [ ] Undertøj og sokker
- [ ] Solhat
- [ ] Let jakke/trøje
- [ ] Regntøj (afhængigt af destination)
- [ ] Sandaler og lukkede sko

### Strand og pool

- [ ] Svømmevinger eller badering
- [ ] Vandsko
- [ ] Dykkerbriller
- [ ] Sandlegetøj

## Teenagere (11+ år)

Teenagere kan ofte selv pakke, men tjek at de har:

- [ ] Oplader til telefon og andre devices
- [ ] Powerbank
- [ ] Solbriller
- [ ] Egen toilettaske
- [ ] Eventuelt egne hovedtelefoner
- [ ] Læsestof eller e-reader

## Håndbagagen – det vigtigste

Alt det der SKAL med i kabinen:

### For voksne
- [ ] Pas og ID
- [ ] Telefon, oplader, powerbank
- [ ] Medicin
- [ ] Et sæt skiftetøj
- [ ] Snacks
- [ ] Underholdning
- [ ] Hovedtelefoner

### For børn
- [ ] Yndlingslegetøj eller bamse
- [ ] Snacks, masser af snacks
- [ ] Tablet/underholdning
- [ ] Skiftetøj (et helt sæt)
- [ ] Tynd trøje (fly er ofte kolde)

## Tips til smart pakning

### Rullemetoden
Rul tøjet i stedet for at folde – det sparer plads og giver færre folder.

### Pakkeposer
Brug pakkeposer eller compression bags til at organisere og spare plads.

### Vask på ferien
Book indkvartering med vaskemaskine og pak mindre tøj med.

### En pose til beskidt tøj
Hav altid en tom pose til beskidt eller vådt tøj.
